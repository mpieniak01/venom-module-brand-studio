# venom-module-brand-studio

Private optional module for Venom: personal brand workflow (discovery -> drafts -> manual publish).

## Local dev
- Keep this module as a separate git repository.
- Register in Venom via `API_OPTIONAL_MODULES=manifest:/home/ubuntu/venom/modules/venom-module-brand-studio/module.json`.
- Enable flags:
  - `FEATURE_BRAND_STUDIO=true`
  - `NEXT_PUBLIC_FEATURE_BRAND_STUDIO=true`
